nginx常用配置

发表于 2016-05-30   |   分类于 Server   |  

说明

nginx的配置是非常丰富的, 本文仅是对常用配置举出一些实例. nginx的配置项和变量均是由模块引入的, 所以本文的结构是按照模块来分的. 并没有介绍全部的模块, 提到的模块里也并没有介绍全部的配置项和变量, 而是仅介绍了经常会用到的, 需更了解更多技术细节请参照官方文档.

Core functionality

ngx_http_access_module

ngx_http_core_module

ngx_http_stub_status_module

ngx_http_auth_basic_module

ngx_http_referer_module

ngx_http_gzip_module

ngx_http_ssl_module

ngx_http_rewrite_module

ngx_http_fastcgi_module

需要通过fastcgi交给后端主机处理的一般都是nginx自己处理不了的动态内容, 但这种方式毕竟没有直接响应的效率高. 所以将处理结果缓存下来, 一定时间内若再有客户端请求相同资源, 则可以直接将缓存结果返回给客户端, 从而大大提高效率.

ngx_http_proxy_module

反向代理时也可以设置缓存, 与fastcgi类似, 就不展开说了.

ngx_http_headers_module

ngx_http_upstream_module

此模块是nginx的一个重要模块, 用来定义上游服务器组成的负载均衡集群. 不过在这里不准备展开, 一是这个属于集群的内容, 在其它介绍集群的文章中提到过一些了; 二是内容比较多不说, 还有一些配置项属于企业版才有的功能. 还是参照官方文档吧.
