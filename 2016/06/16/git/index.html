git服务器搭建

发表于 2016-06-16   |   分类于 Server   |  

参考资料

  • Git教程
  • Pro Git
  • Install a GitLab CE Omnibus package on CentOS 7 (and RedHat/Oracle/Scientific Linux 7)

准备工作

准备一台虚拟机, 均为CentOS-7-x86_64最小化安装, iptables与SELinux均处于关闭状态, 配置好yum源(base和epel). 做好快照, 以便每次实验后快速恢复.

协议类型

  • 本地协议
  • git协议
  • ssh协议
  • http/https协议

本地协议

远程代码仓库放在本地, 其实没什么意义, 所以这种方式应用很少.

git协议

注意: 因为git协议没有认证机制, 通过git协议只能做读操作, 不能做写操作.

ssh协议

http/https协议

http协议是明文传输的, 所以最好使用https协议. 这个实现起来比较简单, 就不演示了.

不过这样使用起来十分麻烦, 一是用户口令认证文件的安全, 二是大量用户管理. 其实可以使用mysql这样的关系型数据库来管理, 不过有更好用的开源解决方案gitlab-ce.

gitlab-ce

For configuration and troubleshooting options please see the Omnibus GitLab documentation
