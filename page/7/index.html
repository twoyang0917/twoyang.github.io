LAMP--CentOS-7平台双机module模型

发表于 2016-04-20   |   分类于 Server   |  

需求

解析

  • 由于httpd是cpu密集型应用，php是io密集型应用，而mariadb既是cpu密集型又是io密集型。所以将httpd+php部署在一台主机上，mariadb部署在另一台主机上。
  • 由于CentOS-7平台提供了以上各应用的rpm安装包，且均包含了各应用的新特性，所以各应用程序可以直接使用yum源安装，而无需通过编译源码来安装。
  • xcache为EPEL源中提供的应用程序，因此需要配置启用EPEL源。
  • 因为需要为phpMyAdmin提供https虚拟主机，而ssl协议是基于IP地址的，所以部署httpd+php的主机必须为两个虚拟主机分别提供IP地址。
